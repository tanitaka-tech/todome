振り返り

日 / 週 / 月 / 年 の 4 粒度で、AI と対話しながら振り返りドキュメントを作成します。 その期間に完了したタスク・KPI 進捗を自動で読み込み、AI が気づきを促す質問を投げかけます。

画面の見方

期間タブ (日 / 週 / 月 / 年) 振り返りの粒度を選択します。タブ切替で対象期間と AI のプロンプトが変わります。
対象日 デフォルトは今日。過去日を指定すると「今日に戻す」ボタンが表示され、ワンクリックで戻せます。 週次は対象日を含む週、月次は対象月を指定します。
振り返りを始める AI セッションを開始。 Did / Learned / Next の 3 項目を AI との対話で埋めていきます。
履歴 (リスト / カレンダー) 完了済みの振り返りの一覧。カレンダービューでは月ごとのヒートマップ風に表示されます。

振り返りの流れ

  1. 期間タブと対象日を選ぶ
  2. 「振り返りを始める」でセッション開始
  3. AI が「この期間で何をしましたか？」「学んだことは？」「次に何をしますか？」と順に質問
  4. 回答を入力すると AI がまとめ、ドキュメントに反映
  5. 最後に日評価 (1〜5) をつけて完了
ヒント: 途中で離脱してもドラフトとして自動保存されます。履歴から再開 / 破棄を選択できます。 詳しい仕様は retrospective-mode-spec を参照してください。
