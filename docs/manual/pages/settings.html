設定

アプリの外観・外部連携・AI ツール権限を設定します。 設定はすべてブラウザ・サーバー側に保存され、リロードしても保持されます。

Appearance (外観)

全 12 テーマ（ダーク 6 + ライト 6）から配色を切り替えられます。 カードをクリックするだけで即座に反映され、選択はブラウザの localStorage に保存されます。

ダーク系 Dark / Midnight / Forest / Sunset / Ocean / Slate
ライト系 Beige / Paper / Mint / Rose / Sky / Sand

GitHub 連携

gh CLI を通じて GitHub アカウントと連携し、任意のリポジトリを「保存先」にできます。 設定後は todome.db が定期的にコミット・プッシュされ、端末間でデータを同期できます。

  • 認証: 未連携時は「連携する」ボタンから gh auth login フローを起動
  • リポジトリ選択: 認証後に自分の管理リポジトリから保存先を選択 (新規作成も可)
  • 同期タブ: 連携済みの場合、プッシュ状況や差分を確認できるタブが表示されます

AI ツール権限

AI アシスタントが使えるツール (TodoWrite / Bash / WebFetch など) を制限できます。 デフォルトでは安全側に振り、ボード更新に必要な最小限だけが許可されています。

注意: Bash 等の外部実行ツールを許可する場合は信頼できる環境でのみ使用してください。 AI 側からの操作はチャット内で確認プロンプトとして表示されます。
